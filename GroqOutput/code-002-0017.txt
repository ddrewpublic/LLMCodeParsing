```
int main()
{
	// Declare variables
    int n,i,j,k,w=0;
    int maxzm,maxbk=0;
    char c1;
    int zimu[26]={0}; // Initialize array to store frequency of each letter
    int lm[100]; // Initialize array to store book numbers
    struct{
    	int num;
    	char a[100];
    }book[1000]; // Declare a struct to hold book information

    // Read the number of books
    scanf("%d",&n);

    // Read book information
    for(i=0;i<n;i++)
       scanf("%d %s",&book[i].num,book[i].a);

    // Count the frequency of each letter in each book
    for(i=0;i<n;i++)
    {
    	for(j=0;book[i].a[j];j++)
    	{
    		for(k=65;k<91;k++)
    		{
    			if(book[i].a[j]==k)
    				zimu[k-65]++; // Increment the count for the current letter
    		}
    	}
    }

    // Find the letter with the maximum frequency
    for(i=0;i<26;i++)
    {
    	if(zimu[i]>maxbk)
    	{
    		maxbk=zimu[i];
    		maxzm=i;
    	}
    }

    // Convert the letter index to a character
    c1=maxzm+65; // ASCII value of 'A' is 65, so subtract 65 to get the index

    // Print the letter with the maximum frequency
    printf("%c\n",c1);

    // Print the maximum frequency
    printf("%d\n",maxbk);

    // Find the books that contain the letter with the maximum frequency
    for(i=0;i<n;i++)
    {
    	for(j=0;book[i].a[j];j++)
    	{
    		if(book[i].a[j]==c1)
    		{
    			lm[w]=book[i].num; // Store the book number
    			w++;
    			continue; // Skip the rest of the loop
    		}
    	}
    }

    // Print the book numbers
    for(i=0;i<w;i++)
    	printf("%d\n",lm[i]);

    return 0;
}
```

This code is designed to find the most frequent letter in a set of books and output the letter and the number of books that contain it. The code first reads the number of books and then reads the information for each book, including the book number and the book title. It then counts the frequency of each letter in each book and finds the letter with the maximum frequency. The code then outputs the letter and the maximum frequency, and finally outputs the book numbers that contain the letter with the maximum frequency. The code uses a struct to hold the book information and uses arrays to store the frequency of each letter and the book numbers. The code uses a nested loop to find the letter with the maximum frequency and a separate loop to find the book numbers that contain the letter with the maximum frequency.
